## 📱 LinkedIn-пост (EN)

> 🎓 New portfolio project: AI Course Curator
>
> Solving a real problem any online school has: students ask the same questions dozens of times a day, eating up curators' time.
>
> What I built:
> • RAG assistant with LangChain + FAISS that answers from course materials only
> • Streamlit UI with student-level selector (beginner / advanced)
> • Prompt architecture with actual level adaptation — not just passing a label, but distinct instruction blocks per level
> • Hard guardrails: no hallucinations, always cite source, no grading or deadline changes
> • JSON request logs as a foundation for topic analytics
>
> Biggest takeaway: in v1 I passed `level` into the prompt as `Level: {level}` — the model ignored it. Only after replacing it with detailed per-level instruction blocks did beginner and advanced responses actually diverge. Small change, big impact.
>
> Stack: Python · LangChain · FAISS · OpenAI · Streamlit
>
> 🔗 Code and details: [github.com/q6066697/ai-course-curator]
>

## 💼 Описание для FL.ru

**Услуга:** Разработка ИИ-ассистента (RAG) на ваших данных

**Описание:**

Создаю end-to-end ИИ-ассистентов с retrieval-augmented generation. Ассистент отвечает только по предоставленной вами базе знаний, ссылается на источник, не выдумывает фактов вне контекста.

Реализованный кейс — ИИ-куратор для онлайн-курса:
- Отвечает на вопросы студентов по материалам лекций
- Адаптируется под уровень (новичок / продвинутый)
- Не выходит из роли при попытках сменить тему
- Логирует все запросы для последующей аналитики

Стек: Python, LangChain, FAISS / Chroma / Pinecone (по требованию), OpenAI / Anthropic / локальные модели.

Готов взять задачу под ключ или присоединиться к команде.
